# CODSOFT Machine Learning Internship

This repository contains the Machine Learning projects completed as part of the CodSoft Virtual Internship Program.

## Projects

### 1. Credit Card Fraud Detection
Developed a machine learning model to identify fraudulent credit card transactions using classification algorithms and data analysis techniques.

### 2. Customer Churn Prediction
Built a predictive model to identify customers likely to leave a subscription-based service using historical customer data.

### 3. Spam SMS Detection
Implemented a Natural Language Processing (NLP) model to classify SMS messages as spam or legitimate.

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn
- NLTK
- Jupyter Notebook
